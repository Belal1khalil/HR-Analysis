# HR Performance & Productivity Analysis Dashboard

## Overview

Interactive Power BI dashboard analyzing employee performance and productivity data to uncover workforce trends, departmental efficiency, and key drivers of high vs. low performance.

## Key Areas of Analysis

- Employee performance distribution across departments and job roles
- Productivity trends and performance scoring
- Workforce demographics segmented by performance level

## Tools & Techniques

- **Power BI**: Interactive dashboard design with slicers and dynamic visuals
- **Power Query**: Data cleaning and transformation
- **DAX**: Custom measures for performance and productivity KPIs

## How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Use slicers to filter by department, role, or performance metrics
